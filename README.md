# Проект

Проект явлется выполнением практических задач по курсу https://stepik.org/course/175243/syllabus.
Каркас приложений генерировал ИИ, а затем он проходил доработку и рефакторинг мной :).

## Подпроекты

1) Json rpc сервер "Спецификация API для управления доступом к секретным данным".
2) Soap сервер "SOAP-сервис для управления задачами (Task Management)".
3) GRPC сервер "gRPC API спецификация для управления информацией о продуктах в интернет-магазине".
4) Rest сервер "RESTful API для управления базой данных о книгах и авторах в библиотеке".
